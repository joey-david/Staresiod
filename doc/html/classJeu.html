Detailed Description

Classe représentant le jeu.

Classe représentant le jeu. Cette classe est utilisée pour créer, initialiser, mettre à jour et afficher le jeu.

La classe gère les éléments du jeu, tels que le vaisseau, le terrain, les power-ups et les astéroïdes.
